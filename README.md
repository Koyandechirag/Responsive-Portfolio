# 🌐 Chirag Koyande – Portfolio

This is my personal **developer portfolio website** built using **HTML, CSS, and JavaScript**.
It showcases my skills, projects, and contact information.

## 🚀 Live Demo

You can view the portfolio here:

🔗 https://koyandechirag.github.io/Responsive-Portfolio/

---

## 📌 Features

* Responsive design (works on desktop, tablet, and mobile)
* Modern UI with animations
* Typing text animation in hero section
* Dark mode support
* Smooth scrolling navigation
* Portfolio project showcase
* Contact form integration with Google Sheets
* Social media links

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**
* **JavaScript**
* **Typed.js** (typing animation)
* **ScrollReveal.js** (scroll animations)
* **Boxicons** (icons)

## ⚙️ How to Run the Project (Linux / Windows / Mac)

1. Clone the repository

```
git clone https://github.com/Koyandechirag/Responsive-Portfolio.git
```

2. Go to the project directory

```
cd Responsive-Portfolio
```

3. Open the project

You can simply open `index.html` in your browser.

Or run a local server:

```
python3 -m http.server
```

Then open:

```
http://localhost:8000
```
